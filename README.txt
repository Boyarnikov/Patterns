Основная идея - игра, управление которой происходит посредством комманд твича
Удачные импликации этой механики можно посмотреть на твиче по тегу Twitch Plays

Юниты:

    В игре основными боевыми единицами являются юниты
    у них есть здоровье, щиты, слоты атаки, защиты, а так же умения ими пользовалься
        Основная боевая единица игроков - солдаты
        У каждого есть имя и управленец - игрок
            "Лидеры" - особые виды солдат, дающие очки инициативы своему отряду
            "Проклятые" - особые виды солдат, имеющие шанс выполнить любое действие дважды. Имеют проклятие
            "Освещённые" - особые виды солдат, имеющие шанс исполнить "освещение", выполняя конкретное действие
        Основная боевая единица врагов - монстры
            "Монстры Лидеры" - особые виды солдат, дающие очки инициативы своему отряду
